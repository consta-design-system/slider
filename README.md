# [Дизайн-система Consta](https://consta.design/) | slider

Компонет слайдера

# Как использовать

## Установите пакет

```sh
# NPM
$ npm install @consta/slider
# Yarn
$ yarn add @consta/slider
```

## Подключите зависимости

Чтобы начать работу, установите библиотеку [`@consta/uikit`](https://www.npmjs.com/package/@consta/uikit) и [настройте тему](http://portal.consta.design/libs/portal/theme-themeabout).

### Можно использовать компоненты

Например, так:

```js
import React from 'react';
```

## Разработка

### Подготовка окружения

Рабочее окружение должно содержать NodeJS и Yarn, необходимые версии можно узнать в файле [package.json](./package.json) в блоке **engines**.

Чтобы установить зависимости, выполните команду:

```sh
$ yarn install
```

### Основные команды

```sh
# Сборка и старт
$ yarn start
# Сборка для production
$ yarn build
# Линтинг всех файлов
$ yarn lint
# Форматирование всех файлов prettier
$ yarn format
# Запуск юнит-тестов
$ yarn unit
# Запуск юнит-тестов, тестирование TS, линтинг файлов
$ yarn test
```
